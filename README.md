Challenges & Solutions

One challenge I faced was balancing the design inspiration with the actual assignment requirements. I wanted the site to look modern and visually appealing, but I also had to make sure it still used semantic HTML, one external CSS file, and a responsive layout that worked across the required breakpoints. Another challenge was making the sidebar and hero section fit properly on different screen sizes. At first, some elements looked too large, the image was pushed down, and the sidebar needed scrolling on smaller widths. I solved this by reducing spacing, resizing cards and images, and adjusting the layout with media queries so the design stayed neat on mobile, tablet, and desktop. I also had to carefully organize button styles and section layouts so the page looked consistent without becoming too crowded.
